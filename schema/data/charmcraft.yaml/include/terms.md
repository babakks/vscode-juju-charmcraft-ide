<a href="#heading--terms"><h2 id="heading--terms">`terms`</h2></a>

**Status:** Optional.

**Purpose:** Lists the terms that any charm user agree to when they're using the charm.

**Structure:** The list of terms:

```yaml
terms:
    - <term>
```

**Example:**

```text
terms:
- Butterscotch is regal
- Cara is adorable
```
